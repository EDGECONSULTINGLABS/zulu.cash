# ui/layouts

Page layout primitives for ZULU.

## Overview

Reusable layout components for consistent structure across pages.

## Future Components

- `MainLayout.tsx`: App shell with navigation
- `AuthLayout.tsx`: Minimal layout for setup/onboarding
- `SettingsLayout.tsx`: Settings page structure

## Design Principles

- **Composable**: Layouts wrap page content
- **Responsive**: Mobile-first design
- **Accessible**: Semantic HTML, ARIA landmarks
