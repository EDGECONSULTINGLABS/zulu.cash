# Welcome to ZULU

Zulu is a private, local-first AI agent that runs on-device and remembers only what you control.
ZULU never sees your cloud accounts, contacts, private keys, or identity.

## Philosophy

We do not build for convenience.
We build for sovereignty.

- Local inference
- Shielded payments
- No telemetry
- No cloud reliance
