# wallet/orchard

Orchard protocol support for ZULU.

## Overview

Orchard is the latest shielded pool in Zcash, offering:
- Enhanced privacy
- Better performance
- Unified addresses

## ZULU Integration

ZULU supports:
- Orchard viewing keys
- Orchard receiver addresses
- Orchard note scanning

## Non-Custodial Design

ZULU uses Orchard for:
- Detecting incoming shielded transactions
- Displaying transaction metadata
- Selective disclosure (user-controlled)

ZULU never:
- Holds Orchard spend keys
- Signs Orchard transactions
- Processes payments
