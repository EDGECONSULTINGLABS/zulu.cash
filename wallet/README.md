# wallet

Zcash-aligned payment verification for ZULU.

- View-only, non-custodial design
- Focused on shielded transaction detection
- No spend keys, no custody, no cloud.

Subfolders:
- `zec/` – ZEC-specific primitives and types
- `lightwalletd/` – integrations with lightwalletd for scanning
- `orchard/` – Orchard-focused logic and docs
