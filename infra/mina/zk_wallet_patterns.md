# ZK Wallet Patterns

## Overview

Zero-knowledge patterns for private wallet operations.

## Patterns

### 1. Prove Balance Without Revealing Amount
```
Prove: balance > threshold
Without revealing: actual balance
```

### 2. Prove Transaction History
```
Prove: made N transactions
Without revealing: transaction details
```

### 3. Prove Compliance
```
Prove: no suspicious activity
Without revealing: full transaction log
```
